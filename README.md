#  A Simple Number Guessing Game (Tkinter GUI) 

##  Game Overview
This game is a number guessing game that began as a console program and evolved into a full Tkinter GUI application.
The player tries to guess a randomly generated number within a range, with real-time feedback.

##  Features
- **Random number** generation between 1 and 20  
- **Tkinter GUI** with entry box and buttons  
- **Best score** tracker (fewest guesses)  
- **Dark mode toggle** (light/dark themes)  
- **Game statistics:** total games played and average guesses per win  
- **Reset button** to start a new round  

##  Technologies Used
- Python 3  
- Tkinter GUI library  
- Git & GitHub for version control  

##  How to Run
Make sure Python 3 is installed on your system.

Run the game from the terminal or VS Code:

```bash
python guessing_game.py
